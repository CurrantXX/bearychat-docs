# Github Webhook设置

1. ## 进入项目的管理页面

   找到你管理的进入项目页面，点击「Settings」

2. ## 添加 Webhook
   点击「Webhooks & Services」-> 「Add webhook」 这时Gtihub会让你输入二次确认密码
3. ## 设置具体的 Webhook 内容
   1. 在「Payload URL」中填入插件webhook url，
   2. 选择希望trigger webhook的event
   3. 勾选「Active」
   4. 点击绿色的「Add webhook」

   非oauth设置就完成了！
