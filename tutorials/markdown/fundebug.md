# BearyChat FunDebug 配置

## 进入 FunDebug 控制台，选择需要项目，点击 `项目设置`

![](/tutorials/image/fundebug_settings.png)

## 在 `报警设置` 选项中，点击 `添加+`

![](/tutorials/image/fundebug_add_alarm.png)

## 选择 `倍洽`

选择 `倍洽`，将 `{{webhookUrl}}` 填入 `配置链接` 中

![](/tutorials/image/fundebug_webhook.png)

## 完成配置

添加完成后，即可在 BearyChat 中收到来自 FunDebug 的报警消息

![](/tutorials/image/fundebug_preview.png)
